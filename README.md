# CMC Go

CMC Go is a map-first coordination app for CMC conference attendance management.

## Docs

- Docs index: [docs/README.md](docs/README.md)
- Contributor + repo guide: [docs/project/REPO_README.md](docs/project/REPO_README.md)
- Coordinator doctrine: [docs/authority/CMC_GO_COORDINATOR.md](docs/authority/CMC_GO_COORDINATOR.md)

## Quickstart

- Install: `pnpm install`
- Local DB + seed (dev only): `pnpm db:setup`
- Dev server: `pnpm dev`
- Tests: `pnpm test`
- E2E smoke: `pnpm e2e`
