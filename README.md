- `.github` navigation: [.github/README.md](.github/README.md)

- Archived/legacy agent docs: [.github/\_unused/README.md](.github/_unused/README.md)

## Quickstart

- Install: `pnpm install`
- Local DB + seed (dev only): `pnpm db:setup`
- Dev server: `pnpm dev`
- Tests: `pnpm test`
- E2E smoke: `pnpm e2e`
